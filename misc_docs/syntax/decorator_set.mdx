The `@set` decorator is used to set a property of an object.

For example:

<CodeTab labels={["ReScript", "JS Output"]}>

```re
type element

@scope("document") @val
external createElement: string => element = "createElement"

@set
external setScrollTop: (element, int) => unit = "scrollTop"

let div = createElement("div")
setScrollTop(div, 100)
```

```js
var div = document.createElement("div")

div.scrollTop = 100
```

</CodeTab>
