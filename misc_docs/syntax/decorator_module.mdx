The `@module` decorator is used to bind to a JavaScript module.

For example:

<CodeTab labels={["ReScript", "JS Output"]}>

```res
@module("path")
external dirname: string => string = "dirname"

let root = dirname("/User/github")
```

```js
var Path = require("path");

var root = Path.dirname("/User/github");
```

</CodeTab>

See the [Import from JavaScript](/docs/manual/latest/import-from-export-to-js#import-from-javascript) section for full details, including other variations of how this decorator may be used.
