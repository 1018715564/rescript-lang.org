# Integer and Float

## Integers

32-bits, truncated when necessary. We provides the usual operations on them: `+`, `-`, `*`, `/`, etc.

### Usage

See [Js.Int](/apis/javascript/latest/js/int) for helper functions.

### Tips & Tricks

**Careful when you bind to JavaScript numbers**! Long ones might be truncated. Bind JS number (especially Dates) as **float** instead.

## Floats

Float requires other operators: `+.`, `-.`, `*.`, `/.`, etc. Like `0.5 +. 0.6`.

### Usage

See [Js.Float](/apis/javascript/latest/js/float).
