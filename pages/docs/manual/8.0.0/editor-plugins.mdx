# Editor Plugins

**Note**: the current editor plugins work with the old `.re` and `.ml` syntax (plus highlighting support for the new syntax for VSCode, Atom and Sublime Text). We'll be releasing a dedicated ReScript plugin in the coming days.

- [VSCode](https://marketplace.visualstudio.com/items?itemName=jaredly.reason-vscode)
- [Atom](https://github.com/reasonml-editor/atom-ide-reason)
- [Vim/Neovim](https://github.com/reasonml-editor/vim-reason-plus)
- [Sublime Text](https://github.com/reasonml-editor/sublime-reason)
- [IDEA](https://github.com/reasonml-editor/reasonml-idea-plugin)
- [Emacs](https://github.com/reasonml-editor/reason-mode) (currently unmaintained)
